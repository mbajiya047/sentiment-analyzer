# Sentiment Analyzer

An AI-powered Sentiment Analyzer built with **Flask**, **TextBlob**, **HTML**, **CSS**, and **JavaScript**. It analyzes text and classifies it as **Positive**, **Negative**, or **Neutral** while displaying sentiment polarity and subjectivity scores.

## Features

- Analyze text sentiment instantly
- Positive, Negative, and Neutral classification
- Displays polarity and subjectivity scores
- Modern responsive UI
- Flask backend with TextBlob NLP
- Easy deployment on Render

## Tech Stack

- Python
- Flask
- TextBlob
- HTML5
- CSS3
- JavaScript
- Gunicorn

## Installation

```bash
git clone https://github.com/mbajiya047/sentiment-analyzer.git

cd sentiment-analyzer

pip install -r requirements.txt

python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
